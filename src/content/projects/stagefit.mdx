## Overview

StageFit is a comprehensive fitness and wellness management system that combines data science with health tracking to provide users with tools to monitor their fitness progress, analyze trends, and make informed decisions about their health journey.

## Key Features

- **BMI Analysis & Health Metrics Dashboard** - Real-time BMI calculation and categorization with visual gauge representation
- **Personalized Diet Planning & Meal Recommendations** - Daily calorie needs calculation with support for multiple dietary preferences
- **Custom Exercise Programs** - Customized workout plans based on fitness goals and health conditions
- **Progress Tracking with Visual Analytics** - Daily progress logging with comprehensive metrics tracking
- **Goal Setting & Achievement Monitoring** - Multiple goal types support with progress visualization
- **Health Reports & Data Export** - Comprehensive health reports with data export functionality

## 🚀 Installation

### 1. Clone the repository:

```bash
git clone https://github.com/GithubMaster-in/StageFit.git
```

### 2. Navigate to the project directory:

```bash
cd StageFit
```

### 3. Install required dependencies:

```bash
pip install -r requirements.txt
```

## ▶️ Usage

Run the application:

```bash
streamlit run app.py
```
